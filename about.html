Acerca del Proyecto

Este sitio web fue creado como parte de nuestro aprendizaje en desarrollo web. El objetivo es practicar HTML, CSS, JavaScript y herramientas de trabajo colaborativo como Git y GitHub, mientras construimos un portafolio básico para cada integrante del equipo.

¿Quiénes somos?

Somos un equipo de 3 estudiantes interesados en la tecnología y el desarrollo de software. Cada uno ha contribuido con secciones distintas del sitio, desde el diseño visual hasta la lógica y organización del contenido.

Objetivos

Nuestro objetivo principal es aprender a trabajar en equipo usando herramientas modernas de desarrollo web. Planeamos seguir ampliando este proyecto con nuevas funcionalidades como formularios, publicaciones dinámicas y diseño responsivo.
